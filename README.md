# Consulta WGH Argentina

Landing page de la **Comisión Consulta** de Women in Global Health Argentina — Consulta Nacional *"Lo que las Mujeres Necesitan"*.

Página HTML self-contained (un solo archivo, sin paso de build). Lista para publicar en GitHub Pages.

## Identidad
- Acento de la comisión: **naranja `#E8512A`**
- Paleta base: midnight `#0D2B35`, aqua `#96D6DF`, lime `#C8E535`, gold `#F5A623`
- Tipografía: Inter

## Cómo editar
- **Contador de respuestas:** en `index.html`, en el `<div id="counter" data-current="1800" data-goal="5000">`. Cambiá `data-current` (respuestas actuales) y `data-goal` (meta, 20000). Se actualiza solo en toda la página.
- **Toolkit de amplificación:** las 10 piezas están en el array `PIECES` dentro del `<script>`. Editá los textos libremente. Usá `[TU LINK]` como marcador del link personalizado.

## Publicar en GitHub Pages
1. Subí este repo a GitHub (usuario `milicaro`).
2. Settings → Pages → Source: `main` / `/ (root)`.
3. La página queda en `https://milicaro.github.io/consulta-wgh/`.
